## 功能组件

### 1. 页面头部工具栏
- **搜索框**: 支持按集群名称、标签搜索
- **状态筛选**: 全部/正常/异常/离线
- **导入集群按钮**: 主要操作入口

### 2. 集群列表表格
| 列名 | 宽度 | 说明 |
|------|------|------|
| 状态 | 60px | 🟢正常 🟡警告 🔴异常 ⚫离线 |
| 集群名称 | 150px | 可点击进入详情 |
| 版本 | 80px | K8s版本号 |
| 节点数 | 60px | 总节点/就绪节点 |
| 就绪率 | 80px | 进度条显示 |
| CPU使用率 | 80px | 进度条显示 |
| 内存使用率 | 80px | 进度条显示 |
| 最后心跳 | 120px | 相对时间 |
| 操作 | 120px | 查看/编辑/删除 |

### 3. 操作按钮
- **查看**: 进入集群详情页
- **编辑**: 编辑集群配置
- **删除**: 删除集群（需确认）
- **终端**: 打开Web Kubectl

## 交互设计

### 状态指示
- 🟢 正常: 集群连接正常，所有节点就绪
- 🟡 警告: 部分节点异常或资源使用率高
- 🔴 异常: 集群连接异常或严重错误
- ⚫ 离线: 无法连接到集群

### 鼠标悬停效果
- 表格行悬停: 背景色变化
- 状态图标悬停: 显示详细状态信息
- 操作按钮悬停: 显示操作说明

### 响应式设计
- 移动端: 隐藏部分列，使用卡片布局
- 平板端: 调整列宽，保持表格布局
